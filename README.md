# c# MediGuide AI

AI-powered healthcare assistant for preliminary medical guidance.

## Features
- Symptom Analyzer
- OCR Report Analysis
- Health Risk Prediction
- Emergency Detection
- Medicine Checker
- Specialist Recommendation

## Tech Stack
- React + Tailwind
- Flask/FastAPI
- OCR + NLP
- SQLite/Firebase

## Installation
Frontend:
npm install
npm run dev

Backend:
pip install -r requirements.txt
python main.pyode-astra-healtcare
